# Retro Fitz

Urban streetwear brand merging old-school vibes with new-age flair. This project is a simple static website built with HTML and CSS.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You don't need any special software to run this project. All you need is a modern web browser.

### Cloning the Repository

To get a copy of the code, you need to clone the repository to your local machine. You can do this by running the following command in your terminal:

```bash
git clone https://github.com/AviOfLagos/retrofitz.git
```

### Previewing the Website

Once you have cloned the repository, you can preview the website by following these steps:

1.  Navigate to the project directory in your file explorer.
2.  Double-click on the `index.html` file.

This will open the website in your default web browser.

## Project Structure

The project has a simple structure:

-   `index.html`: The main landing page.
-   `shop.html`: Placeholder for the shop page.
-   `about.html`: Placeholder for the about page.
-   `contact.html`: Placeholder for the contact page.
-   `privacy.html`: Placeholder for the privacy policy page.
-   `style.css`: The main stylesheet for the website.
-   `script.js`: The main JavaScript file for the website.
